# 00 · MANIFESTO

---

## O nome

**CONCRETO, MENTE E TRÓPICOS**

Não é um slogan. Não é uma tagline. É uma legenda de planta baixa.

É assim que este projeto deve ser lido: como quem decifra uma inscrição
gravada na fachada de um edifício antes de entrar.

Cada palavra carrega um pilar:

**CONCRETO** representa estrutura, engenharia, construção. Tudo aquilo que
suporta outros elementos. Tudo aquilo que não esconde seus materiais. O
brutalismo arquitetônico como princípio organizador do layout, da tipografia,
da composição. Nada é decorativo. Tudo é estrutural.

**MENTE** representa filosofia estoica, clareza, disciplina. A redução como
método. A restrição como liberdade. Sêneca, Marco Aurélio, Cartas a Lucílio.
Não como citação decorativa — como método de organização da informação. Cada
decisão de design é consequência de uma ideia, nunca de um impulso estético.

**TRÓPICOS** representa o Brasil urbano, editorial, concreto. Não o Brasil
turístico, não o carnaval, não a bandeira. É o calor que atravessa o
concreto. É a tinta que descasca. É o papel que amarela. É o scanner que
captura a imperfeição. É a ferrugem que revela o tempo. É São Paulo, não o
cartão-postal.

---

## O que este projeto é

Um portfólio de engenharia de software que se recusa a parecer um portfólio.

É uma publicação.
É um manifesto.
É um arquivo de pesquisa.
É uma revista técnica.
É um caderno de laboratório.
É um documento de arquitetura.

Nunca um template.
Nunca um site.
Nunca uma lista de projetos com thumbnails arredondados.

---

## O que este projeto não é

- Não é um produto SaaS.
- Não é um site de portfolio moderno.
- Não é um cartão de visita digital.
- Não é uma landing page.
- Não é um blog.
- Não é uma rede social.

Se em algum momento da construção ele começar a se parecer com qualquer
uma dessas coisas, recomece.

---

## A missão

Comunicar rigor intelectual através de direção de arte.

O visitante deve sentir que encontrou algo que não esperava encontrar
na web: uma construção, não uma página.

A experiência deve ser semelhante a:

- Entrar no MASP pela primeira vez e ser confrontado pelo vão livre.
- Abrir uma edição antiga da Ray Gun e não saber onde começar a ler.
- Encontrar uma planta baixa esquecida em um arquivo técnico.
- Folhear um livro de engenharia dos anos 1970.
- Visitar o SESC Pompeia e sentir o concreto respirar.

O objetivo não é impressionar.
O objetivo é comunicar que aqui existe pensamento.

---

## A regra de ouro

**Toda decisão visual deve ser consequência de uma ideia.**

Se você não consegue explicar por que um elemento está ali — além de
"porque fica bonito" ou "porque é tendência" — remova-o.

A beleza deste projeto é a beleza da engenharia: cada viga tem função.
