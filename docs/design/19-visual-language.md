CONCRETO

Significa estrutura.

Tudo aquilo que suporta outros elementos deve utilizar
essa linguagem.

Nunca é decorativo.

----------------------------------

PAPEL

Representa conhecimento acumulado.

Toda informação longa deve parecer impressa.

Nunca digital.

----------------------------------

AZUL COBALTO

Representa pensamento.

Nunca representa interação.

Nunca representa sucesso.

Nunca representa links.

----------------------------------

VERDE ÁCIDO

Representa atividade.

Sistema vivo.

Compilação.

Execução.

Cursor.

Terminal.

Jamais deve aparecer em grandes áreas.

----------------------------------

RUÍDO

Representa imperfeição material.

Nunca deve reduzir legibilidade.

Seu objetivo é lembrar que aquilo possui matéria.

----------------------------------

GLITCH

Representa tensão entre homem e máquina.

Nunca é efeito visual gratuito.

Ele aparece somente quando existe conflito.

Ex.:

hover

loading

troca de estado

shader

imagem

Nunca em texto.
